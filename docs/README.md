# FE Interview Prep（KB）

目标：用 6–8 周（不脱产）把**前端基础八股 + coding 手感 + 项目表达 +（轻量）系统设计**打通。

## 使用方法（每天 60–90 分钟）
- Mon–Thu：1 个主题（学→记→练）
- Fri：题库复习 + 中英文口述录音（各 2–4 分钟）
- Sat：Mock（coding 或系统设计）+ 复盘
- Sun：补短板 + 更新周复盘

## 目录
- `_templates/`：统一模板
- `00-Index/`：索引/题库/复习队列
- `01-Core-JS/` 等：主题笔记

> 规则：每篇笔记必须包含 **EN + 中文** 两个版本的 60 秒回答。
